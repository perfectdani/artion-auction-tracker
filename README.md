# Fantom NFT Marketplace - Artion Auction Contract Tracker

A tracker implementation of Auction contract for NFT trading on Fantom Opera network

## Project Setup
```
npm install 
```

## Compile
```
npm run start / npm run dev (nodemon)
```

#### .env file sample
```
NETWORK_RPC=
NETWORK_CHAINID=
DB_URL=
API_ENDPOINT=
CONTRACTADDRESS=
DECIMAL_ENDPOINT=
```
